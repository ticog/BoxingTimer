# Pro Muay Thai Timer

A professional-grade boxing and Muay Thai timer for interval training.

## Features
- **Configurable Phases**: Set custom warm-up, round, and rest times.
- **Visual Cues**: Color-coded backgrounds for each phase (Yellow: Prep, Green: Work, Red: Rest).
- **Audio Alerts**: Professional sound triggers at the start and end of every round.
- **Glassmorphism UI**: A modern, premium design with blur effects and vibrant colors.

## How to Use
1. Open `index.html` in any modern web browser.
2. Use the settings (⚙️) icon to configure your workout.
3. Click "SAVE & RESET" to apply settings.
4. Press the Play button to start your training!

## Project Structure
- `index.html`: Main structure.
- `style.css`: Modern styling and animations.
- `script.js`: Timer logic and sound management.
- `StartSound.mp3`: The round alert sound.
